# Pokemon Gacha

포켓몬 30주년 기념 1~1025 랜덤 뽑기 사이트입니다.
세레비에서 제공하는 포켓몬 이미지를 활용합니다.

## 실행 방법

### 로컬 실행
1. 레포지토리를 클론합니다.
2. `download.py` 스크립트를 실행하여 `images/` 폴더에 포켓몬 이미지(1025장)를 다운로드 받습니다.
```bash
python3 download.py
```
3. `index.html` 파일을 브라우저에서 열거나 로컬 서버를 통해 접속합니다.

### GitHub Pages 배포 (현재 페이지)
github-pages 브랜치 혹은 main 브랜치에서 Settings > Pages 옵션을 통해 무료로 호스팅하실 수 있습니다.
